# StatTracker #
***

## Description: ##

Tracks various statistics over the lifetime of a bukkit minecraft server and saves them in stats.txt

## List of statistics tracked: ##

-  Arrows Fired
-  Blocks Broke
-  Blocks Burned
-  Blocks Placed
-  Eggs Thrown
-  Items Crafted
-  Mobs Killed
-  Player Deaths
-  Players Joined
-  Snowballs Thrown
